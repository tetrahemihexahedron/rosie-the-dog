About this Website

This website was created by me, Sunny Fawcett. As well as being a showcase for the exploits of Rosie the Dog, it is a playground for me to practice modern CSS. The current version of the site uses only vanilla HTML and CSS.

Since the goal is to learn and gain experience, my use of AI has been limited to accepting small carefully reviewed and frequently edited tab completions in Cursor and occasionally using ChatGPT and Cursor’s Plan mode as virtual rubber ducks when evaluating different approaches. My experience is that writing well-organized, easily maintained and extended CSS is not a strength of most AI models. I hope to become a more effective guide for AI agents, and to be able to contribute more to teams of humans, by gaining more hands-on experience with modern CSS features.

Along with consulting MDN and other on-line resources, I’ve watched quite a few YouTube videos, discussed CSS with other developers on dog walks, and am now taking Josh Comeau’s CSS for JavaScript Developers course.
